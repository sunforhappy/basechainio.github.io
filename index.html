Basechain: A Chain of Chains

https://github.com/basechainio Basechain 不仅提供了一条公链，也提供了一个分布式平台的验证，部署和建设生态。他的出现，可以解决区块链的快速迭代，验证，部署。是一条区块链领域的端到端解决方案和开发者社区。而 Basechain 的参与者可以获得不同的区块链项目的体验，参与建设，从中获得回馈。

1. 背景

Basechain 致力于去提供一整套完整的方案去简化区块链想法提出，设计，验证和落地的过程。Basechain 比开发者社区更像是一个宏大的开源公链网络建设计划。BCC (Basechain Chainbuilder Chain)是一套开发工具链。和 hyperledger fabric 不同，他更多的关注在帮助开发者发明与验证新的在不同假设下的分布式系统。Basechain 可以帮助越来越的开发者和普通人去根据自己的商业想法来设计出自己的系统。Basechain 立足于成为区块链领域的GNU Project.

在目前的区块链开发领域，存在着几个问题:

  1. 公链开发门槛高，而所谓的公链的一致性算法在初期得不到有效的验证，一个公链的开发周期通常长达一年甚至若干年。比硬件的开发周期甚至更长，该领域的市场溢价因此过高。而且许多公链开发代码质量堪忧，社区也没有健康的开发讨论。

  2. 缺乏非线性系统的讨论与设计，众所周知，无论是比特币还是最近比较火热的EOS，都不是形式上绝对正确的共识算法，有很多基于社会和组织的假设。因此如何设计，验证这些假设，以及基于这些假设设立环境来检验一个共识算法是否有效。

  3. 合约和贡献证明，是多个商业力图解决的问题，相比较高 TPS，如何去证明用户的其他形式的贡献，比如文件存储，带宽等。以及设计出更具颠覆性的合约框架。如目前的“智能合约”需要所有的节点在输入一致的情况必然得到一致的输出。

Basechain 团队发现了这些问题，并与一些致力于用去中心化技术进行商业实践的团队一道，组成了 Basechain 社区，共同追求 Basecoin 的发展。真正为社区提供一套从想法到落地的去中心化平台。

Basecoin 目的是成为链的链。

2. 概览与架构

区块链简而言之，是一个有着写入规则的分布式数据库。并在特定条件下保证了该数据库的一致性。Basechain 团队在浏览了市面上绝大多数公链项目后，给出了如下的分层:

  • 节点层: 任何分布式系统的基础必然是节点，节点与节点之间的通信构成了节点网络。节点层需要实现的基本功能是:

    • 节点的发现
    • 节点间通讯路由

    在一些情况下，我们需要额外的机制来解决女巫攻击问题。即在对等网络中，单一节点具有多个身份标识，通过控制系统的大部分节点来削弱冗余备份的作用。为了解决女巫供给问题，我们提出了代价函数的概念，即 。 在经典的 PoW 中，由于节点并不会有实质性的投票作用，所以分子为0。从而女巫攻击的代价无限大。而在其他网络中，例如 BFT, 女巫攻击是可能存在的，所以需要增加 Cost Of Creating New Node, 来提高攻击代价，例如 DFINITY 以及采取PoS 机制的分布式系统。但 Basechain 提供的框架并不认为 PoS 不需要考虑女巫攻击问题，而是 PoS 恰好解决了这个问题。

  • 协商层: 协商层是其他区块链公链往往没有的一个特殊层，他是多样化共识算法的支撑。简而言之，节点层通过协商层的抽象来实现不同的共识算法。协商层通常要完成:

    • 随机数协商: 例如 Proof Of Luck 或者 DEFINITY都会要求有一个可被所有节点协商认可的随机数生成机制。
    • 适配协商: 用来和其他已经运行的公链进行数据交换，以用来完成一些特殊的操作。
    • 通讯协商: 如何进行广播，宣告等操作。
    • 加密协商: 协商加密的算法。

  • 共识层: 共识层定义了共识算法以及块数据的范式，是区块链协议的核心部分。在Basechain框架中，允许你构建链结构和DAG结构。

  • 应用层: 通常包含一个有限状态机(智能合约的解释器)。

Basechain 本身是这么的一个公链，但 Basechain 公链的节点层的节点运行了一个被称之为母节点的软件:

母节点: 母节点是 Basechain 节点的子程序，当有人创建了一个 Chain Contract(链合约)，母节点会根据链合约的参数和配置，运行一个新的节点程序，而该节点将会跑新的链。母节点的运行者除了获得跑新链得到的收益之外，也可以通过 Basechain 上链合约规定的激励，来获得额外的收益。

通过链合约，公链开发团队可以让自己的公链快速获得可信的节点网络，也就是共识复制，将 Basechain 已经形成的共识复制给在 Basechain 上共生的其他链。与 EOS 等子链体系不同，Basechain 的链合约可以允许开发者开发符合自己特色的新链。而越来越多的新的链也将让 Basechain 的共识更加可靠。

为了让公链的代币融资和换算更简单，链合约可以嵌入到智能合约内部，当链合约运行时，会自动将智能合约的代币映射为链上的代币。

Basechain 将会成为未来分布式系统世界的核心枢纽。

3.产品和社区

3.1 Basechain 的产品工具

Basechain 在第一阶段准备提供三个工具来实现:

  • Basechain Network: Basechain 网络，作为其他公链项目用来部署，交换自己区块链项目的核心凭条。

  • Base Chainbuilder Tools: 是一套用来生成链合约的工具链，分别在节点，协商，共识以及应用层来根据开发者的需要进行设计。

  • Midgard: Midgard 是一个分布式系统模拟工具，Midgard提供了:

    • 一个分布式系统模拟器，帮你度量在不同网络条件下你的算法的收敛速度。
    • 代币经济模型模拟器，Midgard 会考虑到你代币的经济模型的实际运转情况。
    • 非线性分析工具: 在真实世界里，会有一些政策因素，交易所因素对你的代币和区块链产生影响，因此我们会给出一些非线性分析工具。
      • 激励函数分析
      • 纳什均衡分析
      • 分布度分析
    • Midgard 志愿者计划，midgard 会和 Basecoin Network 联通，Basecoin Network的用户会成为你的测试用户，来帮助你模拟在真实情况下你的经济学模型的假设是否符合你的预期。

我们认为这些工具会极大地减轻开发者的工作量，并让真正有价值的代币的投资流程缩短，投资门槛降低。因为小范围的测试会快速得到一个好的区块链的项目实际用户需求度以及用户增长曲线。而 Basecoin 的用户也获得了极大地回报。因为他们可以参与到很多公链或者DAPP的使用中来。

Midgard 志愿者社区将会成为去中心化应用和商业最前端的支持者。也会成为 Basecoin 最有价值的组成部分之一。

3.2 Basechain 社区和 Basecoin

Basecoin 是 Basechain 流通上的代币，用来维护网络的基本运行以及执行智能合约和链合约:

  • Basecoin 被激励给完成 Basechain-BFT投票的人
  • Basecoin 用来作为执行智能合约的奖励
  • Basecoin 用来作为执行链合约，即你运行一个新链的节点的奖励

无论是智能合约的执行需求还是你需要运行一条自己的链，Basecoin都是你需要的硬通货。我们将通过 Proof Of Propasal 的形式来作为你运行了某个特定链的证明。

4.框架与算法

4.1 Basechain Network 的共识算法

Basecoin Network 是一个针对普通用户和开发者的公链。Basecoin Network 的特点是具有极强的可扩展性。

  • 修改了协商层的广播，采用了“收集者”机制。消息不比发送给每个人，而是发送给收集器，收集器发送给每个人。 如果消息采用加密方式签名，会使用阈值签名的方式使得收集器消息大小从线性减少常量。
  • 使用阈值签名减少通信的总大小。在过去的解决方案中，由于每个都需要一个签名验证来发送 ACK消息给对方，所以每次创建交易，每个节点需要条消息。当节点数变多的时候，会给网络增加巨大的负担。Basecoin Network 的 “收集者”会收集所有的阈值签名，然后再发送消息给每个节点。
  • 采用 Boneh–Lynn–Shacham 签名，在安全性上匹配RSA2048，但是体积更小，速度更快。
  • 使用了乐观执行(optimistic executions)，用乐观执行来进行视图更新(View Change).减少了需要传输的历史状态。

5.1 Newscoin Network

Newscoin Network是一个基于区块链技术的内容经济网络。Newscoin Network的核心主张：把中心化内容平台享有的线上流量红利，归还给内容生产者、分发者和消费者。线上最高频的用户行为是阅读和点击。从门户、搜索到社交媒体时代，每一个普通用户的阅读和点击，造就了以新浪、百度、微博、今日头条为代表的中心化内容平台。但是，过去20年来，内容的生产者、分发者和普通消费者没有得到应有的回报。Newscoin Network认为，用户的每一次阅读和点击行为，都应获得合理的数字资产回报。

Newscoin Network由中天资讯价值联合会（Midheaven Newscoin Association）发起，总部位于中国香港。任何一个线上内容应用，都可以申请接入Newscoin Network。

Newscoin 是 Newscoin Network的催化剂和燃料，是其唯一流动代币。Newscoin Network将把所有接入节点的用户生产、分发和消费行为资产化进而数字化、流通化，并使用 Newscoin（代码NEWS，中文名牛币）作为数字资产奖励。  Newscoin Network提供一个共享平台，在这个共享平台当中有三类角色:

  • 内容的生产方
  • 内容的消费方
  • 内容的分发方

Newscoin Network 或采用区块链技术和其他点对点的平台技术，提供:

  • 内容确权机制: 内容的原创作者通过发布自己创作的内容到 Newscoin Network 上，基于区块链技术保护原创作者的所属权不会被篡改。
  • 作品版权交易: 通过确权，内容的版权以及收益权等更容易地被交换和使用。
  • 用户行为激励: 阅读内容和创作内容本身都应该被激励，同时用户在阅读过程中提供的评价也是应该被奖励的。 …

Newscoin Network为内容的生产方、消费方和分发方提供了大量的工具和激励，帮助他们更方便的通过内容的数字化权力来构造自己的商业，为内容的发展贡献力量。

5.2 巴比塔 Kevin Kelly作为顾问的项目

巴别塔源自圣经《旧约全书》，是人类产生不同语言的起源。在这个故事中，一群只说一种语言的人在“大洪水”之后，决定修建一座城市和一座“能够通天的高塔。上帝见此情形，就把他们的语言打乱，让他们再也不能明白对方的意思，还把他们分散到了世界各地。

过去几千年来，语言差异是信息传播的最大障碍，也是人类社会任何一个共识达成的必过难关。目前世界上现存的语言超过6900种，不同的语言背后，是不同的文化和价值观。唯有重建巴别塔，才能从不同之处求得大同，从大同之中理解不同。

巴别塔是一个基于区块链技术的应用平台，致力于推动全球跨语言内容的发现、翻译和传播。 与其他区块链项目不同，巴别塔将真正通过人工智能和区块链技术提高生产力，也优化生产关系。

中文资料和文化作品的对外输出需求迫切，市场巨大。同时，海外优质内容的输入也存在着无限的潜力。中国、美国以及欧洲也是区块链、人工智能技术和移动互联网发展的主要国家。以网络文学为例：欧美地区可供阅读的网络文学不过50部，翻译人员不到200人。同时存在着以下三个问题： 版权界定模糊 翻译效率低下 商业落地薄弱 巴别塔团队及其合作方在内容的创作、翻译等领域积累了丰富的经验和技术优势，决定以一种非公司化的平台来更好地提供类似网络文学出海的产品。

  • 发现节点：内容的发现者将内容通过区块链进行分享和确权。

巴别塔致力于使用区块链技术来解决版权界定模糊问题，利用人工智能引擎和智能合约来解决翻译效率低下的问题，利用代币激励网络来解决商业落地薄弱的问题。

6.治理委员会

Basechain 为了更好的支持社区发展，将 15% 的代币交由治理委员会。并设计了一套机制来选取治理委员会。

治理委员会由“众议院”和“参议院”组成。采用“代议制度”，“众议院”和“参议院

  • 众议院代表"人民"的意志，其投票权由节点和人数决定，其主要决裁链下事务, 共设 127 席位。
  • 参议院代表“持币者“的意志，其投票权有持币量决定，其主要裁决链上事务, 共设 15 席位。

选举办法:

  • 每届治理委员会任期为 846720 Block
  • 治理委员会选举委员会于每次任期开始提前 120960 Block 开始
  • 治理委员会选举委员由上一届治理委员会中的众议院随机抽选 42 人担任，若不满 42 人的，由项目团队担任选举委员会
  • 为了更好的尊重开发者的利益，每个周期代码贡献度最多的组织和个人可以委托一人直接进入治理委员会的参议院
  • 虚拟选区，为了更好的尊重 Basechain 的社区意愿，治理委员会每次选举发布 42 个有关去中心化商业，哲学，经济以及技术的问题。用来分成 2^42个小区，并使用最小近邻算法合并为127个虚拟选区
  • 虚拟阵营，治理委员会每次选举发布 42 个有关去中心化商业，哲学，经济以及技术的问题。用来分成 2^42个小区，并使用最小近邻算法合并为2个虚拟阵营
  • 众议院和参议院分别由虚拟选区和虚拟阵营进行选举

众议院权力:

  • 决定代码的硬分叉
  • 决定出块时间，奖励
  • 讨论制定出块标准和共识算法

参议院权力:

  • 决定运营资金
  • 仲裁链上纠纷
  • 提供雷电网络通道，即提供快速的支付结算服务，并获取佣金
